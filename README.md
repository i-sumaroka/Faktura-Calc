# FakturaCalc 2026 🇵🇱

Сучасний калькулятор для розрахунку податків JDG (Sole Proprietorship) у Польщі на 2026 рік.

## Особливості
- **Повна підтримка 2026**: ZUS (Pełny, Preferencyйny, Ulga na start).
- **Здоров'я (Health Tiers)**: Три рівні відрахувань для Ryczałt (498 / 830 / 1495 PLN).
- **Валюти**: Робота з PLN, EUR, USD з автоматичним оновленням курсу через NBP API.
- **Ryczałt FAQ**: Вбудований довідник за ставками (2% - 17%) та PKD.
- **Двосторонній розрахунок**: Від Нетто до "На руки" та навпаки (Target Netto).
- **VAT 23%**: Розрахунок повної суми фактури (Gross).
- **Прогноз на рік**: Оцінка нетто-доходу за 12 місяців.

## Як користуватись
Просто відкрийте `index.html` у будь-якому браузері.

## Деплой (GitHub Pages)
1. Створіть новий репозиторій на GitHub.
2. Завантажте файли `index.html` та `styles.css`.
3. Перейдіть у **Settings** -> **Pages**.
4. Оберіть гілку `main` і натисніть **Save**.
5. Ваш сайт буде доступний за адресою `https://[username].github.io/[repo-name]/`.
